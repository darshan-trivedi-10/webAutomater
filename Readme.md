# Web - Automater

### Setup 

1. Clone this repository:
    ```shell
    git clone https://github.com/your-username/website-automator.git
   ```
2. Navigate to the project directory:
    ```shell
    cd website-automator
    ```
3. Install the dependencies:
    ```shell
    npm install
    ```
4. Start the Script
    ```shell
    npm start
    ```
